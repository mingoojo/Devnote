## Routing

> 웹사이트는 url에 따라서 다른 화면을 보여준다.

## HTML DOM API

>

```tsx
function App() {
  const { pathname } = window.location;
  return (
    <div>
      <Header />
      <main>
        {pathname === "/" && <HomePage />}
        {pathname === "/about" && <AboutPage />}
      </main>
      <Footer />
    </div>
  );
}
```
